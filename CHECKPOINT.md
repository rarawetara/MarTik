## Новое в этом чекпоинте

### Дизайн-система
- Единый чёрно-белый стиль (no colors), шрифт Zen Kurenaido
- `design-system.css` — централизованные переменные, каталог-компоненты (`catalog-*`), карточки, кнопки, фильтры
- Унифицированные страницы Гардероб и Красота через `catalog-page / catalog-header / catalog-toolbar / catalog-grid / catalog-card`
- Размер шрифта: базовый 24px, `p` — 15px

### Сегодня (Today)
- Drag-and-drop карточек (`@dnd-kit/sortable`), порядок в `localStorage`
- Мультиселект настроений, одинаковые карточки (вода, задачи, уход, настроение, журнал)
- Журнал: textarea без блокировок, debounce сохранение
- Вода: кнопка «отмена» (−250 мл)

### Задачи
- **Повторяющиеся задачи** — таблица `task_templates` (daily / weekdays / weekends / weekly)
- **Задачи на будущее** — тип `persistent`: появляются каждый день, пока не выполнены; при выполнении шаблон деактивируется
- **Категории задач** — создать/удалить, назначить задаче, фильтр по категории (localStorage)
- **Inline-редактирование** задачи прямо в строке (карандаш → input)
- **Глобальный FAB** — плавающая кнопка задач на всех экранах (AppShell)

### Красота — вкладка Прогресс
- **Категории заметок** — пользовательские категории для фото-заметок, pill-теги
- **Фильтр дневника** — по типу (Лицо/Волосы) и категории
- **Inline-редактирование** заметки в таймлайне (без модала)
- **Дневник-таймлайн** — вертикальный список заметок из прогресс-фото
- **Задачи на будущее** — отдельный блок (localStorage)

### Дневник
- Ежедневный прогресс по 5 столпам (вода, уход, задачи, образ, журнал)
- Цветные кружочки, процент выполнения, трофей за полную неделю

### Главная (Home)
- Все карточки с едиными стилями (`dashboard-card`)
- Drag-and-drop виджетов (`@dnd-kit/core`), layout в `localStorage`
